# Shared Assets

This directory contains shared branding assets used by the web application.

## Directory Structure

```text
assets/
└── favicon.png
```

## Favicon

- File: `favicon.png`
- Format: PNG
- Usage: browser favicon for `apps/web`

The favicon is referenced from `apps/web/index.html` and can also be imported through web build tooling if needed.

## Best Practices

1. Keep source assets versioned in this directory
2. Use optimized PNG files
3. Verify visual clarity at small sizes (16x16 and 32x32)
